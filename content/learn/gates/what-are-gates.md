title: What Are Gates?
description: See how quantum circuits are built from gates on a qubit wire.
---

A **quantum gate** is an operation that changes a qubit's state — like rotating a vector on the Bloch sphere. Gates appear as labeled boxes on a horizontal wire in a **circuit diagram**.

Circuits read left to right: start with |0⟩, apply gates in sequence, then measure. Each gate's output becomes the next gate's input.
