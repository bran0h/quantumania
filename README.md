# Quantumania

Interactive learning platform for quantum computing beginners. Built with Nuxt 4 and Nuxt UI.

## Quick start

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) and navigate to **Learn** to start Chapter 1.

## Scripts

| Command | Description |
|---|---|
| `pnpm dev` | Development server |
| `pnpm build` | Production build |
| `pnpm preview` | Preview production build |
| `pnpm lint` | ESLint |
| `pnpm typecheck` | Vue/TS type checking |
